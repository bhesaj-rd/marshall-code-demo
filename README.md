# Marshall Code Demo

A clean, minimalistic React application with smooth animations and a journey-focused storytelling approach.

## Features

- Modern React with Vite
- Smooth scroll animations
- Parallax effects
- Intersection Observer for scroll-triggered animations
- Fully responsive design
- Clean, minimalistic UI

## Development

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
npm run preview
```
